This is an example of a Xamarin (c#) client connecting to a mormot server.

The mormot server is here:
https://github.com/synopse/mORMot/tree/master/SQLite3/Samples/04%20-%20HTTP%20Client-Server

1. Keep the mormot server running
2. Run the delphi / fpc client once to create one message 
3. Run this app to read the message
